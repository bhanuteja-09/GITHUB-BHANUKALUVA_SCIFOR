4.What is HTML?
HTML stands for Hypertext markup language. It is a standard markup language. 
It is used for developing or creating the web pages and 
It is Used for text formatting and it can formate the text into images and tables and buttons etc.
Hypertext indicates that is used specially for the web browsers.
markup means generally a tag or elements

5.What are the Void Elements in html?
Void elements in html are elements that do not have closing tags means it does not contain the content.
They are self-closing tags and it is also called as singular tags.
 Some common void elements include 
 <br>
 <hr>
 <img>
 <input>
 <link> and etc
